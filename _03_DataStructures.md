# Data Structures Classes (137-206)

This module covers fundamental data structures and their implementations in C#.

## Topics Covered

### Introduction (Class 137)
- Data structures overview
- Time and space complexity basics

### Linked Lists (Classes 138-151)
- Single linked list implementation
- Add, remove, search operations
- IEnumerable and foreach support
- Double linked list
- Improved double linked list with tail pointer

### Queues (Classes 152-156)
- Queue implementation
- Array-based queue
- Linked list-based queue
- Queue operations (enqueue, dequeue, peek)

### Stacks (Classes 157-164)
- Stack implementation  
- Array-based stack
- Linked list-based stack
- Stack applications

### Sets (Classes 165-169)
- Set operations (union, intersection, difference)
- Hash-based sets
- Set implementations

### Hash Tables (Classes 171-179)
- Hash function design
- Collision resolution
- Dictionary implementation
- Hash table operations

### Binary Search Trees (Classes 180-206)
- BST structure and properties
- Insert, search, delete operations
- Tree traversals (inorder, preorder, postorder)
- BST applications
- Balanced trees concepts
