# ASP.NET Web API Classes (312-321)

This module covers building RESTful APIs using ASP.NET Web API.

## Topics Covered

### Web API Basics (Classes 312-314)
- Web API introduction
- HTTP protocol
- REST principles
- Controllers and routing
- Creating a simple API

### HTTP Features (Class 315)
- Query strings
- HTTP headers
- Request/response handling

### Practical Applications (Classes 316-317)
- Chess game API
- Hosting on Azure
- Entity Framework integration

### Advanced Topics (Classes 318-321)
- Logging and monitoring
- Swagger/OpenAPI documentation
- Professional controller design
- Cookies and session management
- Authentication and authorization
- JWT tokens
