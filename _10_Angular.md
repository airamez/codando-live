# Angular Classes (338-353)

This module covers the Angular framework for building modern web applications.

## Topics Covered

### Angular Basics (Classes 338-341)
- Angular introduction and setup
- Components architecture
- Component data binding (one-way and two-way)
- Template syntax
- Forms and data binding demo

### Directives & Control Flow (Classes 342-344)
- Structural directives (*ngIf, *ngFor)
- Modern control flow (@if, @for)
- ng-container

### Component Communication (Classes 345-346)
- ViewChild and template references
- Component lifecycle hooks
- Parent-child component interaction
- Inner/nested components

### Angular Features (Classes 347-349)
- Pipes (built-in and custom)
- Routing and navigation
- HTTP requests with Observables
- RxJS basics

### Advanced Topics (Classes 350-353)
- Services and dependency injection
- Third-party components (PrimeNG)
- Component libraries integration
- Complete application demos
