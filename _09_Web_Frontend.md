# Web Frontend Classes (322-337)

This module covers HTML, CSS, JavaScript, and TypeScript for frontend web development.

## Topics Covered

### HTML & CSS (Class 322)
- HTML structure and elements
- CSS styling and selectors
- Layout techniques
- Responsive design basics

### JavaScript Fundamentals (Classes 323-327)
- JavaScript introduction
- Data types and operators
- Control flow structures
- Functions and arrays
- Exception handling (try-catch)
- Object-oriented programming in JavaScript

### DOM & Events (Classes 328-329)
- Document Object Model (DOM)
- DOM manipulation
- Event handling
- CRUD operations example

### HTTP & Integration (Classes 330-332)
- HTTP requests with Fetch API
- Async/await in JavaScript
- Full CRUD example with API integration
- CORS (Cross-Origin Resource Sharing)

### TypeScript (Classes 333-337)
- TypeScript introduction
- Type system and type safety
- Variables and control flow
- Functions and data structures
- Modules and namespaces
- Object-oriented programming in TypeScript
